The FRHSD Web Design Pathway Program is a rigorous 3-year honors program focused on preparing students for a future career in Web Design, or a related technical field.

Make functional and professional websites, participate in an awesome field trip, earn graduation cords and college credit!

Sophomore Year

Honors Web Design

Learn to create multimedia web pages using emerging technologies. You will learn HTML, CSS, & Photoshop in this course.

Junior Year

Honors Advanced Web Design

Design, develop, & publish projects using leading industry web tools. You will use HTML, CSS, & Javascript in this course.

Senior Year

Honors NJIT Advanced Website Development

The potential to earn college credits! You will develop & maintain the front end of professional web pages for real clients.
